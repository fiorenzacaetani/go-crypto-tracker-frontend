# go-crypto-tracker — Frontend

> **Learning project** — A companion frontend for the [go-crypto-tracker](https://github.com/fiorenzacaetani/go-crypto-tracker) backend, built as part of the same personal Go learning exercise.

> **AI Disclosure** — This frontend was generated by Gemini and then customized by me. I am not a React developer; I built this to have a real UI to connect to my Go backend and see it working end-to-end. The React/frontend code is not representative of my skills — the backend is.

A minimal React dashboard that polls the Go backend every 5 seconds and displays the current Bitcoin rolling average and a live price history chart.

![Screenshot](./screenshot.png)
*(Screenshot — run the full stack to see it live)*

## What it shows

- Current rolling average price, with a color indicator (green/red/neutral) that updates as the price moves
- A line chart of the last 90 price readings fetched from the backend

## Tech stack

| Component | Technology |
|-----------|------------|
| Framework | React 18 + Vite |
| HTTP client | axios |
| Chart | Recharts |
| Backend | [go-crypto-tracker](https://github.com/fiorenzacaetani/go-crypto-tracker) |

## Running locally

The frontend expects the Go backend to be running at `http://localhost:8080`. Start the backend first (see the [backend README](https://github.com/fiorenzacaetani/go-crypto-tracker)), then:

```bash
cd frontend
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.
